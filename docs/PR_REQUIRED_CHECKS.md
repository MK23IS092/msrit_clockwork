# PR Required Checks

Use these as merge gates for this repository:

1. **Backend tests**: `Backend CI` workflow must pass.
2. **Android build**: `Android CI` workflow must pass.
3. **No failing lint diagnostics** in changed files.
4. **`REMAINING_WORK.md` updated** when scope/status changes.
5. **Feature-claim alignment** validated against `docs/FEATURE_MATRIX.md` for any user-visible capability changes.
